Las mejores distribuciones ligeras de Linux para computadoras antiguas

[MISSING TRANSLATION] Adrian Mora (Gadem) 28/08/2022

Breve: No tires tu vieja computadora todavía. Use una distribución de Linux liviana y reviva ese sistema de décadas.

¿Por qué no revivir tu vieja computadora con Linux ? Voy a enumerar las mejores distribuciones ligeras de Linux que puede usar en computadoras antiguas.

Algunas de las distribuciones mencionadas aquí también forman parte de las mejores distribuciones de Linux para principiantes . Entonces, es posible que desee verificar eso si es nuevo en el universo Linux.

Si bien nuestro enfoque está en las computadoras más antiguas, también puede usar la mayoría de estas distribuciones ligeras de Linux en hardware relativamente nuevo. Esto le dará un mejor rendimiento si usa su computadora para un uso intensivo de recursos, como la edición de video en Linux.

Veamos qué distribución ligera de Linux puedes usar.

 

Las mejores distribuciones ligeras de Linux para computadoras portátiles y de escritorio antiguas

Nota: La lista no tiene un orden de clasificación en particular, echa un vistazo a los requisitos mínimos del sistema para elegir uno por ti mismo.

16. Q4OS

Soporte para sistemas de 32 bits: Sí

Q4OS es una distribución basada en Debian que tiene como objetivo proporcionar una experiencia rápida al mismo tiempo que ofrece una opción de 32 bits. De hecho, es una de las mejores opciones para sistemas de 32 bits . Cuenta con Trinity Desktop para la edición de 32 bits y el escritorio KDE Plasma en una imagen de 64 bits.

Puede que no ofrezca la mejor interfaz de usuario, pero es simple y, de hecho, está gritando rápido en hardware antiguo. Puede probar esto fácilmente en su computadora anterior para probarlo.

Requisitos mínimos para Q4OS:

  • RAM: 128 MB (Escritorio Trinity) / 1 GB (Escritorio Plasma)
  • CPU: 300 MHz (Escritorio Trinity) / 1 GHz (Escritorio Plasma)
  • Espacio de almacenamiento: 5 GB (escritorio Plasma) / 3 GB (escritorio Trinity)

15. Slax

Soporte para sistemas de 32 bits: Sí

Slax es una distribución de Linux liviana y realmente portátil basada en Debian que puede usar en una unidad USB sin instalarla.

El tamaño del archivo ISO es de poco menos de 300 MB, lo que lo convierte en una excelente opción para computadoras más antiguas. La interfaz de usuario es simple y utilizable con paquetes preconstruidos esenciales para un usuario promedio. Incluso puede intentar personalizar el sistema operativo y realizar cambios permanentes sobre la marcha si lo necesita.

¡Darle una oportunidad!

Requisitos mínimos del sistema para Slax:

  • RAM: 128 MB (uso sin conexión) / 512 MB (para uso del navegador web)
  • CPU: i686 o más reciente

14. Ubuntu MATE

Soporte para sistemas de 32 bits: Sí

Ubuntu MATE es una impresionante distribución de Linux liviana que se ejecuta lo suficientemente rápido en computadoras más antiguas. Cuenta con el escritorio MATE , por lo que la interfaz de usuario puede parecer un poco diferente al principio, pero también es fácil de usar.

Además del soporte de escritorio, también puedes probarlo en una Raspberry Pi o Jetson Nano .

Requisitos mínimos del sistema para Ubuntu MATE:

  • RAM: 1GB
  • CPU: Pentium M 1,0 GHz
  • Espacio en disco: 9GB
  • Resolución de pantalla: 1024x768

13. Zorin OS Lite

Soporte para sistemas de 32 bits: Sí

Zorin OS es una distribución de Linux basada en Ubuntu. Ofrece una edición lite para computadoras más antiguas que presenta el entorno de escritorio Xfce.

Si tiene un sistema decente (no demasiado antiguo), también puede probar el sistema operativo Zorin normal para ver si se ajusta a su propósito.

Requisitos mínimos del sistema para Zorin OS Lite:

  • RAM: 512 MB
  • CPU: 700 MHz de un solo núcleo
  • Espacio en disco: 8GB
  • Pantalla: resolución de 640 × 480

12. Xubuntu

Soporte para sistemas de 32 bits: Sí

Xubuntu es uno de los sabores oficiales de Ubuntu que presenta el escritorio liviano Xfce.

Lo encontrará fácil de usar y también puede instalarlo en sus computadoras más antiguas sin problemas. Dirígete a su sitio web oficial para descargar el ISO (32 bits/64 bits) que necesitas y comenzar.

Requisitos mínimos del sistema para Xubuntu:

  • RAM: 512 MB (se recomienda 1 GB)
  • Procesador: Pentium Pro o AMD Athlon

11. Linux Mint Xfce

Soporte para sistemas de 32 bits: Sí

Si tiene una configuración de hardware decente (consulte los requisitos mínimos a continuación), la edición Linux Mint Xfce será una excelente opción.

Si bien es una distribución basada en Ubuntu, también cuenta con el escritorio Xfce, lo que lo hace lo suficientemente bueno para algunas computadoras antiguas. Teniendo en cuenta que Linux Mint es una de las mejores distribuciones de Linux , también puede probar otras ediciones (como Cinnamon) disponibles.

Requisitos mínimos del sistema para Linux Mint Xfce:

  • 1 GB de RAM (se recomiendan 2 GB).
  • 15 GB de espacio en disco (se recomiendan 20 GB).
  • Resolución 1024×768

10. Peppermint

Soporte para sistemas de 32 bits: Sí

Peppermint es una distribución de Linux centrada en la nube que no necesita hardware de alta gama. Se basa en Ubuntu y utiliza el entorno de escritorio LXDE para brindarle una experiencia más fluida.

Originalmente creado con el enfoque centrado en la web de las netbooks, Peppermint incluye la aplicación ICE para integrar cualquier sitio web o aplicación web como una aplicación de escritorio independiente.

También encontrará útil la documentación . También existe un foro dedicado para ayudar a solucionar problemas y responder a sus preguntas.

Requisitos mínimos del sistema para Peppermint OS:

  • RAM: 1 GB de RAM (recomendado 2 GB)
  • CPU: Procesador basado en arquitectura Intel x86
  • Espacio en disco: al menos 4 GB de espacio disponible en disco

9. Lubuntu

Compatibilidad con sistemas de 32 bits: Sí (versiones anteriores)

El siguiente en nuestra lista de las mejores distribuciones ligeras de Linux es Lubuntu. Como sugiere el nombre, es miembro de la familia Ubuntu, pero utiliza el entorno de escritorio LXDE/LXQT. Desde Ubuntu 18.10 y superior, encontrará LXQT como el entorno de escritorio predeterminado y podría encontrar LXDE como predeterminado en sus versiones anteriores.

De hecho, es uno de los sabores oficiales de Ubuntu .

Lubuntu admite computadoras más antiguas que han sido enterradas (¡es broma! También puede usar Lubuntu en hardware moderno). Lubuntu es uno de los derivados más ligeros de Ubuntu, por lo que se especializa en velocidad y compatibilidad con hardware antiguo.

Lubuntu tiene menos paquetes preinstalados que consisten principalmente en aplicaciones ligeras de Linux . 

El software y los repositorios son los mismos, por lo que obtendrá todo el software que estaba usando en Ubuntu desde sus repositorios. Sin embargo, sugeriría preferir siempre seleccionar una aplicación que no consuma muchos recursos del sistema.

Requisitos mínimos de hardware para Lubuntu:

  • RAM: 1GB de RAM
  • CPU: Pentium 4 o Pentium M o AMD K8 o superior

8. Linux Lite

Compatibilidad con sistemas de 32 bits: Sí (versiones anteriores)

Como sugiere el nombre, Linux Lite  es una distribución ligera de Linux que no necesita hardware de alta gama para ejecutarla. Incluso un principiante podrá usarlo fácilmente en computadoras más antiguas. Linux Lite se basa en  versiones de Ubuntu LTS (soporte a largo plazo).

Aunque es una distribución liviana, viene con algunas herramientas esenciales.

Por ejemplo, puede encontrar Firefox para navegar por la web, Thunderbird para correos electrónicos, Dropbox para almacenamiento en la nube, VLC Media Player para música, LibreOffice para oficina, Gimp para edición de imágenes y ajustes de Lite para modificar su escritorio. (esto puede cambiar dependiendo de la versión que estés usando).

Teniendo en cuenta que está basado en Ubuntu, tendrá mucho soporte y recursos disponibles en línea, ya que puede seguir los tutoriales de Ubuntu.

Requisitos mínimos de hardware para Linux Lite:

  • RAM: 768 MB de RAM (recomendado 1 GB)
  • UPC: procesador de 1Ghz
  • Pantalla: pantalla VGA con resolución de 1024 × 768 (se recomienda una pantalla VGA, DVI o HDMI de 1366 × 768)
  • Espacio en disco: al menos 8 GB de espacio libre en disco

7. LXLE

Soporte para sistemas de 32 bits: Sí

LXLE es en realidad un giro de la versión LTS de Lubuntu. Ahora que Lubuntu viene con LXQT de manera predeterminada, LXLE es una excelente opción para los usuarios que desean usar el entorno de escritorio LXDE .

A pesar de ser una distribución ligera de Linux, LXLE intenta proporcionar una interfaz de usuario intuitiva y atractivos para la vista. El sistema está ajustado para mejorar el rendimiento y viene con una amplia gama de aplicaciones ligeras instaladas de forma predeterminada.

Requisitos mínimos de hardware para LXLE:

  • RAM: 512 MB (recomendado 1 GB)
  • CPU: Pentium 3 (recomendado Pentium 4)
  • Espacio en disco: 8GB

6. CrunchBang++

Compatibilidad con sistemas de 32 bits: Sí (versiones anteriores)

CrunchBang++ también se conoce como CBPP o #!++ o CrunchBang Plus Plus. Crunchbang++ es el clon de Crunchbang Linux que ha sido descontinuado.

CrunchBang ++ es compatible con computadoras antiguas y se ejecuta sin ningún problema. CrunchBang++ se basa en  Debian 10 con una interfaz de diseño minimalista.

Algunas de las aplicaciones predeterminadas en Crunchbang++ al momento de escribir esto eran Geany IDE, el emulador de terminal Terminator, Thunar File Manager, Gimp para edición de imágenes, visor de imágenes Viewnior, VLC Media Player para música, software de grabación de CD/DVD Xfburn, etc. .

Puede que no sea el mejor para todos los usuarios, pero puede probarlo y verlo usted mismo.

Requisitos mínimos de hardware para CrunchBang++:

  • RAM: 1GB de RAM
  • CPU: Pentium 4 o Pentium M o AMD K8 o superior

5. Bodhi Linux

Compatibilidad con sistemas de 32 bits: Sí (versiones anteriores)

Otra distribución ligera de Linux: Bodhi Linux, que da vida a PC y portátiles más antiguos. Bodhi Linux es bastante conocido por su enfoque minimalista y el soporte para hardware de gama baja.

No cuenta con muchas cosas preinstaladas; por lo tanto, notará que el tamaño del archivo ISO será inferior a 1 GB.

La presencia de Moksh Desktop hace que Bodhi Linux sea una opción decente para configuraciones de hardware más antiguas al tiempo que brinda una buena experiencia de usuario.

Requisitos mínimos de hardware para Bodhi Linux:

  • RAM: 256 MB de RAM
  • CPU: 1,0 GHz
  • Espacio en disco: 5 GB de espacio en disco

4. antiX Linux

Soporte para sistemas de 32 bits: Sí

antiX es una distribución ligera de Linux basada en Debian Linux. Si buscas algo que no incluya systemd , esta es una gran opción.

antiX también usa el administrador de ventanas icewm para mantener el sistema funcionando en hardware de gama baja. No tiene mucho software preinstalado, por lo que el tamaño del archivo ISO es de alrededor de 700 MB. Siempre puede descargar e instalar más software más tarde si tiene acceso a una conexión a Internet activa.

Requisitos mínimos de hardware para antiX Linux:

  • RAM: 256 MB de RAM
  • CPU: sistemas PIII
  • Espacio en disco: 5 GB de espacio en disco

3. SparkyLinux

SparkyLinux es otra distribución liviana pero, al mismo tiempo, también apunta a las computadoras modernas.

Dependiendo de lo que necesite, encontrará dos variantes de SparkyLinux. Uno basado en la versión estable de Debian y el otro basado en la rama de prueba de Debian. Así que puedes optar por cualquiera que creas conveniente.

Además de las variantes, también encontrarás diferentes ediciones de ISO para descargar. Por ejemplo, una edición de escritorio LXQT, una edición GameOver con cosas preinstaladas, etc.

Puede dirigirse a su página de descarga y hacer clic en las versiones "Estables" o "(Semi-)Rolling" para encontrar todas las ediciones enumeradas.

Requisitos mínimos de hardware para SparkyLinux:

  • RAM: 512 MB
  • CPU: Pentium 4 o AMD Athlon
  • Espacio en disco: 2 GB (Edición CLI), 10 GB (Edición Home), 20 GB (Edición GameOver)

2. Puppy Linux

Compatibilidad con sistemas de 32 bits: Sí (versiones anteriores)

Puppy Linux se puede iniciar en vivo con un CD/DVD/USB.

1. Tiny Core

Probablemente, técnicamente, la distribución más ligera que existe. Sin embargo, no es una distribución de Linux completa para un usuario de escritorio promedio.

Tiny Core simplemente incorpora el núcleo fundamental de un sistema operativo que incluye el kernel y el sistema de archivos raíz. En otras palabras, presenta la base de un sistema operativo de escritorio.

Si simplemente desea que un sistema se inicie junto con una conexión a Internet por cable, puede comenzar a usarlo. Pero, no debe esperar un soporte de hardware adecuado listo para usar.

Entonces, si eres alguien que sabe cómo configurar o compilar las herramientas necesarias con Tiny Core Linux para que sea una experiencia de escritorio completa, puedes intentarlo.

Requisitos mínimos de hardware para Tiny Core Linux:

  • RAM: 64 MB (se recomiendan 128 MB)
  • CPU: i486DX

Conclusión

La mayoría de las distribuciones de Linux deberían ser fáciles de instalar en computadoras más antiguas sin problemas. Si está interesado en una buena experiencia de usuario, una interfaz de usuario fácil de usar y estabilidad, hay muchas opciones en nuestra lista.
