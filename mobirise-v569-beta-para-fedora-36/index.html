Mobirise v5.6.9 Beta para Fedora 36 (Distros RPM)

Adrian Mora (Gadem) Published on junio 6, 2022


Qué hay de nuevo: * Correcciones para el fondo de YouTube * Correcciones para desplazamiento suave y enlaces de anclaje. * Solución para una situación en la que hay un espacio al comienzo del título de la página * Menú arreglado para Safari en Mobirise 5 * Hemos agregado una nota para no cambiar el nombre de index.html * El botón Exportar sitio se renombró a Exportar proyecto * Arreglos para la carga del proyecto
Fuente: https://forums.mobirise.com/discussion/29931/mobirise-v5-6-9-beta
Si tiene algún problema con sus proyectos, puede restaurarlos a través del Historial de proyectos: https://mobirise.com/help/project-history-465.html
Enlace de Descarga: https://itsyour.site/mobirise/mobirise-5.6.9-60.x86_64.rpm
