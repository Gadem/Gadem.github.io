Paper es una elegante aplicación para tomar notas para Linux

Adrian Mora (Gadem)
12/06/2022
Actualizado en 12/06/2022

Si está buscando una aplicación para tomar notas para Linux, querrá consultar Paper .

Paper es una aplicación GTK moderna que se presenta a sí misma como una 'hermosa aplicación para tomar notas para GNOME', ¡y lo es! Estar basado en Markdown significa que la aplicación es compatible con el formato de texto, la adición de enlaces, el resaltado de secciones de texto, la inserción de bloques de código y más, desde el primer momento.

Ahora, me clasifico más como un tomador de notas casual. Utilizo aplicaciones como esta para recopilar ideas que tengo para futuras publicaciones, comandos de terminal que debo recordar y... Bueno, mantente al tanto de mi colección de juguetes He-Man.

Necesidades básicas para tomar notas, por así decirlo. No estoy en la universidad haciendo un título que necesita citas, referencias, cálculos, clasificación jerárquica y otras opciones "avanzadas". Para tomar notas serias, lo mejor es un servicio, una aplicación y/o una plataforma más establecidos.

Pero para las necesidades diarias? El papel es el billete.

Funciones en Paper, la aplicación GTK Notes

Paper te permite crear cuadernos (carpetas, por así decirlo) donde viven las notas. Cuando crea un nuevo cuaderno en Paper (a través del menú o el atajo shift++ ctrl) n, puede nombrarlo, establecer un color para denotarlo (que se muestra en la barra lateral) y elegir el esquema de 'nombramiento' que aparece en la portada ( o utilice un icono en su lugar).

Si observa la captura de pantalla a continuación, verá varios cuadernos visibles en la franja en el lado izquierdo de la ventana de la aplicación. Las 'cubiertas' de estos cuadernos son todas diferentes. Algunos tienen nombres, algunos tienen iconos, todos tienen diferentes colores.

Además de poder crear cuadernos separados, también puede crear fácilmente notas dentro de los cuadernos.

Paper admite Markdown de forma nativa (aunque los botones WYSIWG para acciones comunes como negrita, tachado, encabezados, enlaces, etc. se encuentran en la barra de herramientas en la parte inferior de la ventana de edición de notas), por lo que puede agregar la mayoría de los formatos de texto en línea con los elementos representados en la pantalla cuando esté completo. (similar a Typora y Mark Text ).

El color de la opción de formato de resaltado cambiará entre las notas, ya que se deriva del color de la cubierta del cuaderno.

Naturalmente, esta aplicación admite el modo oscuro (aunque globalmente, no como una configuración específica de la aplicación) y tiene una barra lateral plegable que le permite simplificar la interfaz de usuario. Juntas, estas opciones pueden ayudarlo a concentrarse únicamente en la tarea de tomar notas.

En cuanto a la configuración, la aplicación le permite elegir una fuente para usar en las notas y cambiar la ubicación de almacenamiento de las notas guardadas (lo que significa que, en teoría, puede sincronizarlas a través de la nube entre dispositivos). Si está en un dispositivo OLED, también hay una opción para habilitar un modo negro "verdadero" (aunque esto solo está activo cuando el modo oscuro está configurado en GNOME globalmente).

Organizar notas es simple gracias a arrastrar y soltar notas. Puede reordenar las libretas en la barra lateral de esta manera, o arrastrar y soltar notas de una libreta a otra. También puede mover notas usando el menú contextual.

Actualmente no puede agregar etiquetas o etiquetas a las notas y filtrarlas de esa manera, pero la aplicación cuenta con búsqueda de texto completo, por lo que, si lo ha escrito, ¡debería poder encontrarlo!

Si bien no iré tan lejos como para llamar a Paper un rival de Evernote y amable (al menos no todavía), es, en lo que respecta a las aplicaciones para portátiles de Linux, una alternativa sólida. A Linux no le faltan aplicaciones para tomar notas, y esta trae una interfaz de usuario moderna de estilo 'libadwaita' a la mesa, es compatible con dispositivos móviles y admite Markdown de forma nativa.
