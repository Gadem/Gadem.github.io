aaPanel, una alternativa a cPanel gratuito y Open Source

Adrian Mora (Gadem)

En el siguiente artículo vamos a echar un vistazo a aaPanel. Este es un panel de control de servidor gratuito y de código abierto, que es simple y liviano. Con este software podremos administrar servidores en un entorno de alojamiento web. Aquí tendremos la posibilidad de administrar y controlar dominios, SSL, bases de datos MySQL y otras funciones del alojamiento web. En las siguientes líneas, vamos a ver cómo instalar este software en Ubuntu.

aaPanel también se ejecuta con buenos resultado en servidores de bajos recursos. Este software nos ofrecer a los usuarios un panel de control simple pero poderoso, el cual nos va a permitir administrar el servidor web a través de una GUI (interfaz gráfica de usuario) basada en la web.

aaPanel proporciona la función de instalación con un solo clic, y gracias a esta dispondremos de la posibilidad de instalar con un solo clic el entorno de desarrollo y el software LNMP / LAMP. Con este instalador automático, se pueden instalar diferentes aplicaciones con un solo clic. Su objetivo principal es ayudar a los usuarios a ahorrar tiempo de implementación, para que se centren en su propio proyecto.

Características generales de aaPanel

  • El concepto de desarrollo modular que utiliza aaPanel, permite que los usuarios tan solo instalemos las extensiones que nos interesen.
  • Permite la monitorización de los recursos. Podremos supervisar la ocupación de los recursos de nuestro servidor en tiempo real, lo que nos permitirá comprender mejor su capacidad de carga.
  • El panel proporciona una puerta de enlace antispam gratuita, Nginx WAF, recordatorio de inicio de sesión SSH y extensión de seguridad del firewall del sistema, entre otras cosas.
  • Dispone de un potente editor en línea. aaPanel añade un poderoso editor en línea, combinado con el administrador de archivos.
  • Además de todo esto, este software nos va a permitir trabajar con; dominios web, dominios DNS, dominios de correo, bases de datos, CRON, directorios de usuarios y más.

Este programa cuenta con una versión gratuita y otras de pago. En la versión gratuita encontraremos un buen puñado de opciones disponibles, aun que en las versiones de pago, como es lógico, encontraremos más opciones. Todas las características disponibles de la versión gratuita se pueden consultar en la página web del proyecto.
