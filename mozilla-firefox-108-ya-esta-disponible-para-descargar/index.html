Mozilla Firefox 108 ya está disponible para descargar

Adrian Mora (Gadem)
17/12/2022
Actualizado en 17/12/2022

Una nueva versión del navegador web Firefox está disponible para descargar.

Mozilla Firefox 108 es el último lanzamiento previsto para este año. Cuenta con un pequeño conjunto de mejoras modestas diseñadas para mejorar la usabilidad, la seguridad y la estabilidad del navegador cuando se usa para acceder a las principales plataformas de escritorio.

La nueva característica "titular" es la compatibilidad con la API WebMIDI . Esto permite que Firefox (más específicamente, las cosas que se ejecutan dentro de Firefox) se conecten e interactúen con dispositivos MIDI (interfaz digital de instrumentos musicales) conectados a una PC. No todos los dispositivos MIDI son instrumentos musicales, por lo que la función también tiene aplicaciones no melodiosas.

Firefox 108 también es mejor en el manejo de caracteres que no son ASCII al guardar e imprimir formularios PDF; agrega la combinación de teclas shift+ escpara abrir Process Manager (una gran utilidad para ver qué pestañas usan más recursos); y gana soporte para bucles de video continuos en sitios/reproductores que lo soporten.

El navegador puede manejar correctamente imágenes utilizando perfiles ICCv4 y permite importar mapas de forma predeterminada para permitir que las páginas web "controlen el comportamiento de las importaciones de JavaScript".

En el lado de Linux, hay correcciones de errores y mejoras de usabilidad, incluida una refactorización del bloque del controlador WebRender para usar una política de 'permitir de forma predeterminada'. Esto puede habilitar WebRender en más dispositivos Linux que antes. Como existe la posibilidad de problemas, Mozilla planea comenzar a bloquear los dispositivos "malos" conocidos en lugar de permitir los "buenos".

Mozilla Firefox 108 es un software gratuito de código abierto. Puede descargar el lanzamiento desde el FTP de Mozilla para todas las plataformas compatibles ahora mismo o esperar a que la actualización se implemente a través de su canal de lanzamiento preferido a partir del martes 13 de diciembre.
