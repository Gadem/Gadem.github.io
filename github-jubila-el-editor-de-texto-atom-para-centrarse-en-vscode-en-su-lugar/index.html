Github jubila el editor de texto Atom  para centrarse en VSCode en su lugar

Adrian Mora (Gadem)
09/06/2022
Actualizado en 09/06/2022

¿Alguien por ahí todavía usa el editor de texto Atom de GitHub?

Si es así, tengo malas noticias para impactar: ​​¡se está descontinuando! Oh, lo sé: precisamente nadie que lea esto está sorprendido por la noticia, aunque sospecho que más de uno se entristecerá.

GitHub fue adquirido por Microsoft hace unos años. Dado que Microsoft tiene su propio editor de texto basado en Electron en VSCode, era probable, si no inevitable, que se eligiera uno y el otro fuera enlatado.

No para disminuir Atom de ninguna manera, pero VSCode de los dos editores es, con mucho, el más popular y ampliamente utilizado. Diablos, incluso yo uso VSCode de vez en cuando (aunque yo mismo prefiero el nuevo editor de texto de GNOME).

“Hoy, estamos anunciando que vamos a cerrar Atom y archivaremos todos los proyectos bajo la organización el 15 de diciembre de 2022”, ha anunciado GitHub . Atom ha seguido recibiendo actualizaciones de mantenimiento y seguridad en los últimos años pero, como señala GitHub, el "desarrollo de características significativas" se ha estancado.

Por desgracia, y lo que es igual de importante, también lo ha sido la participación de la comunidad en el proyecto, el elemento vital de cualquier código abierto.

Como resultado, GitHub dice que ha decidido "... retirar Atom para promover nuestro compromiso de llevar el desarrollo de software rápido y confiable a la nube a través de Microsoft Visual Studio Code y GitHub Codespaces".

Atom no vivirá como editor de texto, pero aún deja un legado bastante grande, a saber, Electron. Sí, el marco de habilitación de aplicaciones multiplataforma que muchos odian por su naturaleza de agregar recursos está en uso en cientos de millones de usuarios de computadoras todos los días, pero sus raíces están en Atom. De hecho, Electron se llamó originalmente 'Atom Shell'.

¿Y ahora qué?

Bueno, si usted o alguien que conoce está usando Atom, no necesita entrar en pánico, todavía no .

Atom seguirá funcionando bien en lo previsible. Pero durante los próximos seis meses, los usuarios verán notificaciones en la aplicación de que la aplicación está por salir. El 15 de diciembre de 2022, los (muchos) repositorios relacionados con Atom en Github se archivarán y el sitio web oficial de atom.io (presuntamente) quedará suspendido.
